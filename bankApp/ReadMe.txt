This is a simple "bank" app that runs in the command line
Written in Java

TO RUN:

1. download files to your computer
2. store Account.java and BankApp.java in the same folder
3. compile both classes (javac *.java)
4. run BankApp (java BankApp)
5. follow prompts

NB!!!
NOT INCLUDED: datatype verification.
Inputs in menu's should be single-digit int's (1, 2, 3, etc...)
Program isn't made to handle invalid datatypes (eg: String when it expects an int)
